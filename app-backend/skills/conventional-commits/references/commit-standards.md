# Commit Standards Reference

Types:

```text
feat
fix
docs
test
refactor
chore
build
ci
perf
style
revert
```

Common scopes:

```text
auth
users
health
api
infra
terraform
docs
deps
ci
prisma
logger
```

Before committing, inspect status and diff. Do not commit secrets, generated state, build output, dependency folders, or ignored local agent folders.

Use a body when the reason is not obvious from the summary.
